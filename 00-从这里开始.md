# 乔帮主第二大脑配置包

> 目标：让 Claude Code 从零开始，帮你建立一套可运行、可回写、可迭代的个人 AI 工作台。
> 适用对象：想把散乱资料变成 AI 能读懂的经验资产的人。

---

## 乔帮主欢迎语

你好，我是乔帮主。

这份配置包，是给有经验的人准备的第一块 AI 工作台地基。

你不用先成为程序员，也不用把所有工具都学明白。你只需要把这份文件交给 Claude Code，让它按步骤帮你安装、连接、验证。等这套第二大脑跑起来，你的网页剪藏、课程资料、外网素材、创作产物，就不再是一堆散乱文件，而会慢慢变成 AI 能读懂、能调用、能复用的经验资产。

先别急着追工具。我们先让 AI 认识你。

---

## 配置包总链路

```text
00 从这里开始（本文件）
    ↓
01 连接 GitHub
    ↓
02 连接 Obsidian 第二大脑
    ↓
03 连接 NotebookLM 课程素材工厂
    ↓
04 生成教学驾驶舱
    ↓
回写 Obsidian，沉淀执行记录
```

---

## 每一步做什么？

| 步骤 | 内容 | 完成后你得到什么 |
|---|---|---|
| 00 | 了解全貌 | 清晰的目标和边界 |
| 01 | 连接 GitHub | GitHub CLI 登录、测试仓库、GitHub Pages 验证 |
| 02 | 连接 Obsidian | 测试 vault、MCP 连接、Web Clipper 采集规则 |
| 03 | 课程素材工厂 | NotebookLM notebook、简报、信息图、讲义、测验 |
| 04 | 教学驾驶舱 | 一个可展示成果的 HTML 页面 |

---

## 怎么用？

把下面这句话交给 Claude Code：

```text
请读取乔帮主第二大脑配置包，并按步骤帮我建立 AI 工作台。
```

Claude Code 会按顺序读取本目录下的文件，一步步执行。

---

## 注意事项

- 遇到需要浏览器登录的地方，Claude Code 会暂停等你操作。
- 已有配置不会直接覆盖，会先确认。
- 本配置包不产生大文件，产物主要是 Markdown 和 HTML。
