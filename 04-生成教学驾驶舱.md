## 进阶：部署到 GitHub Pages

如果想把教学驾驶舱发布到互联网：

1. 把 `index.html` 提交到 GitHub 仓库。
2. 在仓库设置中启用 GitHub Pages。
3. 等待 1-2 分钟后访问 `https://你的用户名.github.io/仓库名/`。

具体命令：

```bash
cd /Users/qiaozhanglong/Digital_Brain/乔帮主第二大脑配置包/GitHub能力包
git add index.html
git commit -m "更新教学驾驶舱"
git push origin main
```

然后启用 Pages：

```bash
gh api repos/{owner}/{repo}/pages --method POST -F 'source[branch]=main' -F 'source[path]=/'
```
